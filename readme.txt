**Beaver Builder Templates by WPZOOM** contains a collection of pre-designed templates for Beaver Builder page builder.

== Description ==

This plugin provides a collection of custom [**Beaver Builder** page builder](https://www.wpbeaverbuilder.com/) templates that can make your job extremely easy to create a beautiful interface of the pages or posts on your WordPress site (especially the accessibility ready themes by [WPZOOM](https://www.wpzoom.com/)).

It is very easy to use and you do not need to write any code. To contribute improve this plugin, please access the [GitHub repository](https://github.com/wpzoom/beaver-builder-templates-by-wpzoom).

The plugin works great with [**Beaver Builder Addons by WPZOOM** plugin](https://wordpress.org/plugins/wpzoom-addons-for-beaver-builder/).

= Features =

* Beautiful pre-designed templates
* Possibility to extend the templates for specific WordPress Theme

== Installation ==

1. Install the Beaver Builder Templates by WPZOOM either via the WordPress plugin directory or by uploading the files to your server (see instructions below).

1. Unzip the plugin download file and upload `beaver-builder-templates-by-wpzoom` folder into the `/wp-content/plugins/` directory.
2. Activate the plugin through the *"Plugins"* menu in WordPress.
3. Now, whenever you edit a page or post with Beaver Builder (you need to have the plugin installed and activated), the page builder interface will contain additional sections of pre-designed templates. ([See how to add a layout template to your page](https://docs.wpbeaverbuilder.com/beaver-builder/layouts/templates/add-a-layout-template-to-your-page))


== Frequently Asked Questions ==

= Who should use the Beaver Builder Templates by WPZOOM? =

The Beaver Builder Templates by WPZOOM is a collection of unique and creative templates that help build beautiful pages and posts on a website. Therefore, it is certainly an asset for all those who love Beaver Builder.

= What are the requirements to use the Beaver Builder Templates by WPZOOM? =

You only need to have the [**Beaver Builder** page builder](https://www.wpbeaverbuilder.com/) and [**Beaver Builder Addons by WPZOOM**](https://wordpress.org/plugins/wpzoom-addons-for-beaver-builder/) plugins installed and activated first.

= What themes does the Beaver Builder Templates by WPZOOM work with? =

The Beaver Builder Templates by WPZOOM is built to work great with all the themes.

= Will the Ultimate Addons slow down my website? =

Absolutely no! The Beaver Builder Templates by WPZOOM only serves additional custom templates to Beaver Builder page builder interface, so it is not slowing down your website at all. Therefore, it will not affect your website speed and performance in any way.

= Can I set it up somehow? =

This is a simple plugin that only adds custom collections of designs into Beaver Builder page builder and there is really no need for any settings page.

If you want, you can modify the plugin with code using action and filter hooks provided. More information on [GitHub repository](https://github.com/wpzoom/beaver-builder-templates-by-wpzoom)
